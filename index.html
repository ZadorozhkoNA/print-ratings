Инструкция по пользованию сервисом

  1. Сформировать в электронном журнале список оценок. Скачать файл-excel «Текущая_успеваемость_ хх.хх_хх.хх.хххх_хххххх.xls».
  2. САМЫЙ ВАЖНЫЙ ПУНКТ! Открыть файл в Excel и сохранить в формате .csv (разделители - запятые).
  3. Загрузите файл (.csv) через форму в верхней части экрана.
  4. Сервис обработает файл: удалит лишние строки и сформирует карточки с оценками для каждого ученика. Результат будет предствален на экране.
  5. Внимание! Перед распечаткой результатов в настройках браузера рекомендуется проверить установки принтера. Оптимальные настройки:
    • выбрать нужный принтер;
    • размер бумаги - А4;
    • ориентация листа - книжная;
    • поля - минимальные. (Если пункт «минимальные» отсутствует, то установите по 1 см с каждой стороны);
    • верхние и нижние колонтитулы - отключить;
    • масштаб - 100%;
  6. Распечатайте файл используя ссылку сервиса «Распечатать» или печать из настроек браузера. Нормальное расположение карточек: в два столбца, 4 - 6 на листе.
  7. Проявите находчивость, озадачьте кого-нибудь аккуратно разрезать листы на отдельные карточки.
